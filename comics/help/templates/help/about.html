About the open source project

Functionality

comics is a web comics aggregator. Out of the box it can crawl and archive about two hundred comics every day.

The comics are made available through an easy to use web interface were users can build personalized collections of their favorite comics, and then read them on the site or in their feed reader of choice.

Development

comics is licensed under the GNU Affero General Public License version 3. This means that anyone using the software must share their local improvements with their users.

Development of the comics software is coordinated through GitHub. The project has extensive documentation available at comics.readthedocs.org.

Feedback to the administrator of this specific comics installation may be submitted through the feedback page.

Technology

comics makes use of the Python programming language, the Django web framework, the jQuery JavaScript library, the Bootstrap web design toolkit, and icons by Font Awesome.

All output should adhere to the HTML5, CSS, and Atom web standards.
